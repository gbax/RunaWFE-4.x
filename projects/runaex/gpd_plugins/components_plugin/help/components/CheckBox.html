Флажок

Задает значение "true" или "false" объекту
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Значение по умолчанию Задание значения, выставляемого в флаге по умолчанию при открытии формы добавления. Доступны три варианта: выбрано, не выбрано и из базы данных.
Копировать из (если значение по умолчанию "Из базы данных") Если выбрано значение по умолчанию "Из базы данных", то в поле необходимо ввести ссылку на колонку таблицы, в которой хранится флаг, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
