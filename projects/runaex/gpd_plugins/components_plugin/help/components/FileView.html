Просмотр файла

Задает просмотр файла, загруженного в базу данных
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ширина просматриваемого изображения Задание ширины просматриваемого изображения. Задается в пикселях. Используется только в случае просмотра файла, содержащего изображение.
Наименование колонки для подписи Название атрибута сущности, в котором храниться подпись просматриваемого файла. Используется в случае участия компоненты просмотра файла в проверке ЭЦП. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Значение по умолчанию Задание значения по умолчанию в компоненте просмотра файла. Ссылка на колонку таблицы, в которой хранится файл, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
