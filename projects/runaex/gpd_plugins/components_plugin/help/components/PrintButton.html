Кнопка печати

Задает печать объекта по шаблону или без
Параметр Описание
Наименование кнопки Отображаемое пользователю наименование.
Идентификатор печатаемой таблицы Параметр таблицы, данные которой будут распечатаны.
Наименование шаблона печатной формы Наименование файла с шаблоном отчета, созданного в программе iReport.
Хранимые процедуры Наименование хранимой процедуры шаблона отчета. Вводится в формате: "имя_хранимой_процедуры(1-параметр,2-параметр,...,n-параметр)". Если параметров нет, то должны быть указаны пустые скобки.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Ширина в пикселях Задание ширины в пикселях отображаемой на форме кнопки.
