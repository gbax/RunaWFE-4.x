Меню навигации

Задает переход на узел действия
Параметр Описание
Заголовок Отображаемый пользователю заголовок меню.
Узлы дерева Задание узлов-действия, на которые будет осуществляться переход из меню навигации. Узлы вводятся в формате: Отображаемое пользователю наименование узла дерева,Наименование родительского узла дерева, Наименование узла-действия, на который осуществляется переход. Разделяются знаком ";". Если необходимо задать узлы на одном уровне, то вместо наименования родительского узла дерева ничего не указывается.
Видимость Параметр, управляющий видимостью меню навигации. Доступны три варианта задания видимости: true, false и значение переменной.
