Компонента проверки ЭЦП

Позволяет проверить ЭЦП сохраненной формы
Параметр Описание
Таблица Название сущности, в которой хранятся данные подписанной формы. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Колонка для сохранения данных Название атрибута сущности, в котором хранятся данные подписанной формы. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Колонка для сохранения подписи данных Название атрибута сущности, в котором хранится подпись данных формы. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
