Группа переключателей

Выбор значения из группы
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ссылка на колонку Ссылка на колонку справочника группы переключателей. Задается в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Является обязательным Задание обязательности заполнения поля ввода. Задается тремя значениями: не обязательно, обязательно, и правило Groovy.
Порядок сортировки Задание порядка сортировки значений переключателей.
Значение по умолчанию Задание типа значения, выставляемого в группе переключателей по умолчанию при открытии формы добавления. Доступны три варианта: вручную, из базы данных и исполнить код Groovy. Если выбран тип "Вручную", то необходимо ввести любое значение из справочника группы переключателей. Если же выбран тип "Из базы данных", то в поле необходимо ввести ссылку на колонку таблицы, в которой хранятся данные группы переключателей, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения. Если же выбран тип "Исполнить код Groovy", то в качестве значения по умолчанию будет подставляться результат выполнения кода Groovy.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
