Загрузка файла

Загружает файл в базу данных
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Является обязательным Задание обязательности загрузки файла. Задается тремя значениями: не обязательно, обязательно, и правило Groovy.
Наименование колонки для подписи Название атрибута сущности, в котором будет храниться подпись файла. Используется в случае участия компоненты загрузки файла в ЭЦП. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Значение по умолчанию Задание значения по умолчанию в компоненте загрузки файла. Ссылка на колонку таблицы, в которой хранится файл, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
