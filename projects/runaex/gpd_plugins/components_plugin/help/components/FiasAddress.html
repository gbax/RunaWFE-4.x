Адрес

Выбор адреса из системы ФИАС(КЛАДР)
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Минимальный уровень Задание уровня, с котрого начнется поиск адресных объектов.
Максимальный уровень Задание уровня, которым завершится поиск адресных объектов.
Предустановленный фильтр Задание значения адресного объекта, ограничивающего поиск. Допустимо указание значения ниже минимального уровня, но не выше максимального.
Является обязательным Задание обязательности заполнения поля ввода. Задается тремя значениями: не обязательно, обязательно, и правило Groovy.
Искать только по актуальным данным Задание типа поиска адресных объектов. Доступны два варианта: да и нет.
Колонка для даты, используется только для не актуальных данных Колонка сохранения прошлой даты, на которую адресные объекты были актуальны. Параметр обязателен для заполнения в случае, если в поле "Искать только по актуальным данным" выбрано значение "нет". Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Колонка для сохранения полного адреса Колонка сохранения найденного адресного объекта. Параметр обязателен для заполнения в случае, если в поле "Искать только по актуальным данным" выбрано значение "нет". Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Значение по умолчанию из базы данных Задание значения по умолчанию в поле ввода. Ссылка на колонку таблицы, в которой хранится адрес, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
