Генератор номеров записей

Генерирует номер сохраняемой записи таблицы
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Имя счётчика Задание имени счетчика.
Шаблон Задание шаблона генерируемого номера записи. В шаблоне обязательно должен присутствовать символ "%"(генерируемый номер) и любые другие символы. Пример шаблона: Номер %.
Генерировать при Задание условия генерации номера записи. Доступны два варианта: при отображении и при сохранении.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
