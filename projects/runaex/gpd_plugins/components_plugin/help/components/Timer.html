Таймер

Отображает сколько прошло времени с определенной даты
Параметр Описание
Колонка с датой старта Название атрибута сущности, в котором хранится дата. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Шаблон Задание шаблона отображения таймера. В шаблоне обязательно должны присутствовать символы: "DD", либо "HH", либо "MM", либо "SS" и любые другие символы. Примеры шаблона: DD-HH-MM-SS,HHч MMм SSс
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
