Скрытое поле

Позволяет сохранить в базу данных скрытое на форме значение
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ссылка на колонку Ссылка на справочник, из которого будет подставляться значение по умолчанию. Задается в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Тип значения по умолчанию Задание типа значения, сохраняемого в скрытом поле по умолчанию. Доступны три варианта: вручную, текущее время и ФИО пользователя.
Значение Если выбран тип "Вручную", то необходимо ввести любое значение из справочника, при условии, что заполнено поле "Ссылка на колонку", если же это поле не заполнено, то необходимо ввести любое произвольное значение. Если же выбран тип "Текущее время" или "ФИО пользователя", то поля "Ссылка на колонку" и "Значение" заполнять не нужно.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
