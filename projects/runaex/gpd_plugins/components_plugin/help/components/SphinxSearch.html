Компонента поиска(SphinxSearch)

Поиск объекта по нескольким колонкам из разных таблиц
Параметр Описание
Таблица Название сущности, в которую будет сохраняться результат поиска. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Наименование индекса Задание наименование индекса. Наименование не должно содержать ключевых слов "index" или "src", а также знаков препинания, кроме символов "-" и "_".
Использовать существующий индекс Доступны два варианта: да и нет.
Индексируемые колонки Задание ссылок на колонки, по которым будет осуществляться поиск. Вводятся в формате: Схема.Таблица.Колонка или Таблица.Колонка, если указана схема по умолчанию. Разделяются знаком ";".
Отображаемые колонки Задание отображаемых в результате поиска колонок. Задается в формате: Наименование колонки, ссылка на отображаемую колонку. Разделяются знаком ";". Ссылка задается в формате: Схема.Таблица.Колонка или Таблица.Колонка, если указана схема по умолчанию.
Идентификатор колонок Задание колонок, в которые будет сохраняться результат поиска. Задаются в формате: Ссылка на таблицу, по которой осуществляется поиск, наименование колонки сохранения результата поиска. Разделяются знаком ";". Ссылка вводится в формате: Схема.Таблица или Таблица, если указана схема по умолчанию.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
