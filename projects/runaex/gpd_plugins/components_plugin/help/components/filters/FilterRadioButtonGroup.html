Фильтр (группа переключателей)

Фильтрует записи в таблице по конкретному значению группы переключателей
Параметр Описание
Колонка Название атрибута сущности, в котором хранятся фильтруемые данные группы переключателей. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ссылка на колонку Ссылка на колонку справочника фильтра группы переключателей. Задается в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Идентификатор, указанный в компоненте таблицы Параметр таблицы, в которой хранятся фильтруемые данные группы переключателей.
Значение по умолчанию Задание значения, выставленного по умолчанию при открытии формы поиска.
Порядок сортировки Задание порядка сортировки значений переключателей.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
