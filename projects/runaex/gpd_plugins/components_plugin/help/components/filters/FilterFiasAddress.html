Фильтр (адрес)

Фильтрует записи в таблице по конкретному адресу из системы ФИАС(КЛАДР)
Параметр Описание
Колонка Название атрибута сущности, в котором хранятся фильтруемые данные адреса. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Идентификатор, указанный в компоненте таблицы Параметр таблицы, в которой хранятся фильтруемые данные адреса.
Минимальный уровень Задание уровня, с котрого начнется поиск адресных объектов.
Максимальный уровень Задание уровня, которым завершится поиск адресных объектов.
Предустановленный фильтр Задание значения адресного объекта, ограничивающего поиск. Допустимо указание значения ниже минимального уровня, но не выше максимального.
Искать только по актуальным данным Задание типа поиска адресных объектов. Доступны два варианта: да и нет.
Колонка полного адреса Колонка, в которой хранится искомый адресный объект. Параметр обязателен для заполнения в случае, если в поле "Искать только по актуальным данным" выбрано значение "нет". Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
