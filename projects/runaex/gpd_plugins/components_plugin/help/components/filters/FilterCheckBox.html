Фильтр (флажок)

Фильтрует записи в таблице по конкретному значению флажка
Параметр Описание
Колонка Название атрибута сущности, в котором хранятся фильтруемые данные флажка. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Идентификатор, указанный в компоненте таблицы Параметр таблицы, в которой хранятся фильтруемые данные флажка.<
Значение по умолчанию Задание значения, выставляемого в фильтре флага по умолчанию при открытии формы поиска. Доступны два варианта: выбрано и не выбрано.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
