Фильтр (список с поиском)

Фильтрует записи в таблице по конкретному значению из справочника списка с поиском
Параметр Описание
Колонка Название атрибута сущности, в котором хранятся фильтруемые данные списка с поиском. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ссылка на колонку Ссылка на колонку справочника фильтра списка с поиском. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Идентификатор, указанный в компоненте таблицы Параметр таблицы, в которой хранятся фильтруемые данные списка с поиском.
Значение по умолчанию Задание значения, сохраняемого в поле ввода по умолчанию.
Связанное поле Ссылка на колонку, куда будут сохраняться данные другого фильтра списка с поиском, от которого зависит текущий фильтр. Используется в случае зависимых фильтров списков с поиском. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Связанная таблица Ссылка на таблицу справочника фильтра списка с поиском, от которого зависит текущий фильтр. Вводится в формате: Схема.Таблица. Таблица выбирается из существующей модели БД, заданной в файле конфигурации БД приложения. Используется в случае зависимых фильтров списков с поиском.
Наименование колонки связи Наименование колонки, присутствующей в таблице справочника текущего фильтра списка с поиском и содержащего идентификатор записи из справочника фильтра списка с поиском, от которого зависит текущий список. Используется в случае зависимых фильтров списков с поиском. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Связующая таблица Ссылка на таблицу, связывающая таблицу справочника текущего фильтра списка с поиском и фильтра списка с поиском,от которого он зависит. Используется в случае зависимых фильтров списков с поиском. Вводится в формате: Схема.Таблица. Таблица выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Порядок сортировки Задание порядка сортировки записей.
Подсказка к полю Текст подсказки, отображаемой в поле ввода.
Поддержка голосового ввода Задание поддержки голосового ввода значения в поле ввода.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
