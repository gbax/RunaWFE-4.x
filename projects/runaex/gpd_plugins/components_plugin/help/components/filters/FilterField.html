Фильтр (текст)

Фильтрует записи в таблице по конкретному строковому значению текстового поля
Параметр Описание
Колонка Название атрибута сущности, в котором хранятся фильтруемые данные текстового поля. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Формат поля Задание формата фильтра текстового поля. Доступны два варианта: текстовое и текстовое(многострочный режим).
Регулярное выражение Задание регулярного выражения для вводимого значения в поле ввода.
Максимальная длина Задание максимальной длины вводимой строки в поле ввода.
Идентификатор, указанный в компоненте таблицы Параметр таблицы, в которой хранятся фильтруемые данные текстового поля.
Сравнение Задание сравнения, на основе которого будет осуществляться фильтрация. Доступны следующие варианты сравнения: равенство, больше чем или равно, больше чем, меньше чем или равно, меньше чем. Последние четыре сравнения применимы только к числовым данным текстового поля.
Значение по умолчанию Задание значения, отображаемого по умолчанию в поле фильтра при открытии формы поиска.
Подсказка к полю Текст подсказки, отображаемой в поле ввода.
Поддержка голосового ввода Задание поддержки голосового ввода значения в поле ввода.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
