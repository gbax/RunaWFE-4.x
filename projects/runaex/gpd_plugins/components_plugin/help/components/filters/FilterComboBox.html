Фильтр (список)

Фильтрует записи в таблице по конкретному значению из справочника списка
Параметр Описание
Колонка Название атрибута сущности, в котором хранятся фильтруемые данные списка. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ссылка на колонку Ссылка на колонку справочника фильтра списка. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Идентификатор, указанный в компоненте таблицы Параметр таблицы, в которой хранятся фильтруемые данные списка.
Значение по умолчанию Задание значения, сохраняемого в поле ввода по умолчанию.
Порядок сортировки Задание порядка сортировки записей.
Поддержка голосового ввода Задание поддержки голосового ввода значения в поле ввода.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
