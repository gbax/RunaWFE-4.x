Компонента подписи и сохранения данных

Позволяет подписывать форму и сохранять данные на ней
Параметр Описание
Наименование кнопки Отображаемое пользователю наименование.
Таблица Название сущности, в которую будут сохранены данные формы. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Колонка для сохранения данных Название атрибута сущности, в который будут сохранены данные формы. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Колонка для сохранения подписи данных Название атрибута сущности, в который будет сохранена подпись данных формы. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
