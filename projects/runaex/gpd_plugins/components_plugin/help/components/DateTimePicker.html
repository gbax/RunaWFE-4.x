Дата и время

Задает дату в формате даты, либо даты и времени
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Является обязательным Задание обязательности заполнения поля ввода. Задается тремя значениями: не обязательно, обязательно, и правило Groovy.
Подсказка к полю Текст подсказки, отображаемой в поле ввода.
Тип Задание типа сохранения данных. Доступны два варианта: дата и дата и время.
Значение по умолчанию Задание типа значения, сохраняемого в поле ввода по умолчанию. Доступны три варианта: вручную, из базы данных и исполнить код Groovy. Если выбран тип "Вручную", то необходимо ввести любое значение в формате даты или даты и времени. Если же выбран тип "Из базы данных", то в поле необходимо ввести ссылку на колонку таблицы, в которой хранится дата или дата и время, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения. Если же выбран тип "Исполнить код Groovy", то в качестве значения по умолчанию будет подставляться результат выполнения кода Groovy.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
