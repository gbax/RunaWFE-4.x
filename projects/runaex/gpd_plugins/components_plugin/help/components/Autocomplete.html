Список с поиском

Поиск значения в справочнике по введенным в поле ввода символам
Параметр Описание
Колонка Название атрибута сущности, в который будут сохранены данные. Выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Ссылка на колонку Ссылка на атрибут сущности, по которому будет осуществляться поиск. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Является обязательным Задание обязательности заполнения поля ввода. Задается тремя значениями: не обязательно, обязательно, и правило Groovy.
Подсказка к полю Текст подсказки, отображаемой в поле ввода.
Связанное поле Ссылка на колонку, куда будут сохраняться данные другого списка с поиском, от которого зависит текущий список. Используется в случае зависимых списков с поиском. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Связанная таблица Ссылка на таблицу справочника списка с поиском, от которого зависит текущий список. Вводится в формате: Схема.Таблица. Таблица выбирается из существующей модели БД, заданной в файле конфигурации БД приложения. Используется в случае зависимых списков с поиском.
Наименование колонки связи Наименование колонки, присутствующей в таблице справочника текущего списка с поиском и содержащего идентификатор записи из справочника списка с поиском, от которого зависит текущий список. Используется в случае зависимых списков с поиском. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Связующая таблица Ссылка на таблицу, связывающая таблицу справочника текущего списка с поиском и списка с поиском,от которого он зависит. Используется в случае зависимых списков с поиском. Вводится в формате: Схема.Таблица. Таблица выбирается из существующей модели БД, заданной в файле конфигурации БД приложения.
Порядок сортировки Задание порядка сортировки записей.
Поддержка голосового ввода Задание поддержки голосового ввода значения в поле ввода.
Значение по умолчанию Задание типа значения, сохраняемого в поле ввода по умолчанию. Доступны три варианта: вручную, из базы данных и исполнить код Groovy. Если выбран тип "Вручную", то необходимо ввести любое значение из справочника списка с поиском. Если же выбран тип "Из базы данных", то в поле необходимо ввести ссылку на колонку таблицы, в которой хранится список с поиском, и значение которой проставляется по умолчанию при переходе из таблицы на форму добавления. Вводится в формате: Схема.Таблица.Колонка. Колонка выбирается из существующей модели БД, заданной в файле конфигурации БД приложения. Если же выбран тип "Исполнить код Groovy", то в качестве значения по умолчанию будет подставляться результат выполнения кода Groovy.
Видимость Параметр, управляющий видимостью элемента. Доступны три варианта задания видимости: true, false и значение переменной.
Редактируемость Параметр, управляющий редактируемостью элемента. Доступны три варианта задания редактируемости: true, false и значение переменной.
