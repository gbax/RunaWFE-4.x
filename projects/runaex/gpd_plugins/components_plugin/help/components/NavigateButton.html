Кнопка навигации

Задает переход на узел-действия
Параметр Описание
Наименование кнопки Отображаемое пользователю наименование
Наименование узла-действия Наименование узла действия, на который будет осуществляться переход при нажатии на кнопку навигации.
Действие кнопки Доступны следующие действия: добавление, редактирование, другое.
Идентификатор таблицы Параметр таблицы, при указании которого кнопка навигации будет заблокирована до тех пор, пока не будет выделена какая-либо запись в таблице.
Код Groovy Код Groovy, выполняемый перед действием кнопки.
Видимость Параметр, управляющий видимостью кнопки действия. Доступны три варианта задания видимости: true, false и значение переменной.
Ширина в пикселях Задание ширины в пикселях отображаемой на форме кнопки.
